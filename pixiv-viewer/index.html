後端 Worker 部署與網域鎖定說明
  1. 登入 Cloudflare Dashboard → Workers & Pages → Create Worker，貼上本專案的 worker.js 後 Deploy。
  2. 把取得的網址（如 https://xxx.workers.dev）填入本檔開頭的 WORKER_URL 常數，重新上傳到 GitHub Pages 即完成。
  3. 想避免額度被其他網站盜用：到 worker.js 開頭把 ALLOWED_ORIGINS 的 "*" 改成你的網域即可。

臨時切換後端可在網址加上 ?worker=https://…，會記住於此瀏覽器。
